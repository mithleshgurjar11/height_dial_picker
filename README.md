## ✨ Features

- 📏 Smooth dial-based height selection
- 🔄 ft / cm unit toggle
- 🎨 Fully customizable UI (colors, range)
- ⚡ Lightweight & no extra dependencies
- 📱 Easy to integrate in any Flutter app
